## 📦 Manual Install

```bash
git clone git@github.com:jofr27/Angry_Penguin.git
cd Angry_Penguin
chmod +x install.sh
./install.sh
```

---

## 🎨 Features

* 🐧 Custom *Angry Penguin* theme
* 🖼️ Integrated Papirus-Dark icon base
* 🎯 Deep-orange folder color scheme
* ⚙️ Omarchy-ready structure
* 🧩 Clean fallback icons for consistency

---

## 🧰 Requirements

* Omarchy
* Hyprland
* `gtk-update-icon-cache`
* (Optional) `yay` + `papirus-folders`

## 🚀 After Install

1. Open **Omarchy Theme Selector**
2. Apply: `angry-penguin`
3. Set icon theme: `Angry-Penguin`
4. Restart session or refresh file manager

---

## 🛠️ Troubleshooting

* Icons not updating?

  * Restart session or file manager
* Theme not showing?

  * Check: `~/.config/omarchy/themes/`
